Linux

Linux介绍：

​ linux is not unix

操作系统：

​ 一套管理硬件和软件的程序

​ 意义：

​ 服务

linux特点：
  1. 模块化程度
  2. 源码公开
  3. 广泛的硬件支持
  4. 安全性及可靠性好
  5. 具有优秀的开发工具
  6. 有很好的网络支持及文件系统支持
  7. 与unix完全支持

​

字符界面命令提示符：

​ 主提示符：

​ [用户@主机名 工作目录]

​ 辅助提示符：

​ #：管理员

​ $：普通用户

目录结构：
目录名称 详细解析 备注
bin 存放普通用户可使用的命令文件 root用户也可使用
sbin 存放仅超级（root）用户可使用的命令文件 普通用户不可以使用
usr/bin 是从bin软连接过来
usr/sbin 是从sbin软连接过来
boot 存放系统启动的一些列文件，如grub，grub2等等
dev 系统的设备文件目录 如：硬盘，光驱等
etc 系统的配置文件目录 如：etc/rc,etc/sysconfig/network-scripts
home 普通用户的家目录 如：使用useradd添加的普通用户，就存放在该目录下
lib 32位系统的库文件 很多命令都依赖库文件
lib64 64位系统的库文件 ，从CentOS7开始，Linux没有32位了
media 媒介目录 如插入u盘，会在该目录下显示
mnt 临时挂载的目录 可临时挂载光驱，新增硬盘等等
opt
proc 系统启动的一些进程，每个进程都有一个pid，pid下，存放这进程的详细信息 ls -l /proc/281查看
root root用户的家目录
run 进程运行时产生的临时文件 重启后，文件就会消失
srv service服务产生的文件
sys 系统内核相关的文件 平时不会动里面的东西
tmp 系统的临时文件目录，权限很大，任何用户都可以访问
usr Unix System Resource，并不是user的缩写 后面一些常见的服务，apache，mysql等服务，会放在usr/local下
var 某些大文件的溢出区，比方说各种服务的日志文件 var/log/message，存放系统日志文件

文件操作命令：

pwd

​ 用途：查看当前工作目录

cd

​ 用途：转到指定目录

​ 语法：cd 目录

ls

​ 用途：罗列出所有当前目录下的所有文件夹

​ 语法：ls [选项]

​ 选项：

​ -l ：显示文件的详细信息（可以简写”ll”）

​ -R：

​ -a：显示当前文件夹的所有文件包括隐藏文件

​ -h：格式化显示文件信息

mkdir

​ 用途：创建目录

​ 语法：mkdir [选项] 目录名

​ 选项：

​ -p：递归创建

rmdir

​ 用途：删除目录

​ 语法：rm [语法] 目录

touch

​ 用途：创建文件

​ 语法：touch 文件名

rm

​ 用途：删除文件

​ 语法：rm [选项] 文件/目录

​ 选项：

​ -r：删除文件夹

​ -f：强制删除

cp

​ 用途：复制文件

​ 语法：cp [选项] 源文件 目标文件

​ 选项：

​ -r：递归复制整个目录树

mv

​ 用途：剪切文件

​ 语法：mv [选项] 源文件目录 目标文件目录

​ 选项：

​ -f：若目标已存在，则不询问直接覆盖

​ 注意：目标文件目录不指定文件名，就是默认源文件名

rm

​ 用途：删除文件

​ 语法：rm [选项] 要删除的文件

​ 选项：

​ -i：删除文件的时候提醒用户确认

​ -r：删除目录

​ -f：强制删除文件不提醒用户

find

​ 用途：搜索文件

​ 语法：find 目录 [选项] 文件名

​ 选项：

​ -name：按文件名字查找

​ -size：按文件大小查找

​ -user：按文件所属者查找

​ -type：按文件类型查找

​ d：目录

​ f： 文件

​ -i：不区分大小写

​ 通配符：

​ *：匹配任意字符

​ ?：匹配任意一个字符

​ []：匹配括号中的任意一个字符

more

​ 用途：全屏方式分页显示内容

​ 语法：more 文件路径

​ 交互操作方法：

​ Enter：向下逐行滚动

​ Space：向下翻一屏幕

​ b：向上翻一屏幕

q：退出

按q键退出

cat

​ 用途：查看文件内容

​ 语法：cat 文件名

tac

​ 用途：倒序查看文件内容

​ 语法：tac 文件名

vi

​ 用途：创建或修改文本文件

​ 语法：vi 文件路径

​ 操作：

​ a\i\o：进入输入模式

​ esc：进入命令模式

​ z:进入末行模式

​ w：保存文件

​ q!：强制退出编辑

​ wq：强制保存并退出

vim

​ 用途：创建或修改文本文件

​ 语法：vi 文件路径

​ 操作：

​ ctrl+f：下一页

​ ctrl+b：上一页

​ 数字0：将光标移动到当前行最前面

​ $：将光标移动到当前行最后面

​ G：光标移动到文件的最后一行

​ nG：光标移动到文件的n行

​ gg：光标移动到文件的第一行

​ 搜索：

​ /abc：搜索abc

​ n：下一个

​ N：上一个

​ 复制、删除：

​ dd ：删除当前行

​ u：回退

​ yy：复制当前行

​ nyy：5yy

​ p：粘贴

压缩解压命令

gzip：

​ 用途：制作压缩文件、解开压缩文件

​ 语法：gzip [选项] 文件名

​ 选项：

​ -9：高压缩比，躲在创建压缩包时用

​ -1：表示压缩最快

​ -d：用于解开已经压缩的文件

tar

​ 用途：制作归档文件、释放归档文件

​ 语法：tar [选项].. 归档文件名 源文件目录

​ 选项：

​ -c：创建.tar格式的文件

​ -x：解开.tar格式的文件

​ -v：显示详细过程

​ -f：使用归档文件（放在选项的最后一位）

​ -z：解开.tar.gz格式的文件

用户与权限

w

​ 用途：查看当前用户的信息

whoami

​ 用途：查看当前用户的用户名

useradd

​ 用途：创建用户

​ 语法：useradd 用户名

​ 选项：

​ -g：创建用户名时并添加到一个用户组

​ 例：useradd student -g teacher

userdel

​ 用途：删除用户

​ 语法：userdel 用户名

passwd

​ 用途：修改密码

​ 语法：passwd 用户名 [选项]

​ 选项：

​ -r：删除用户所有信息

su

​ 用途：切换用户

​ 语法：su 用户名

chmod

​ 用途：更改文件/目录权限

​ 权限值：

​ r：读权限； 权限值：4

​ w：写权限； 权限值：2

​ x：执行权限； 权限值：1

​ - ：没任何权限； 权限值：0

​ 语法：chmod 权限值 文件/目录

例：

​

chown

​ 用途：修改文件所有者和所在组

​ chown 用户名:用户组 文件/目录

​ chown 用户名 文件/目录

​ chgrp 用户组 文件/目录

挂载：

mount

​ 挂载：

​ 指的就是将设备文件中的顶级目录连接到 Linux 根目录下的某一目录（最好是空目录），访问此目录就等同于访问设备文件。

​ 语法：mount /dev/cdrom /mnt

管道

​ 一个命令的执行结果作为下一个命令的参数

​ 语法：

​ 命令1

​ ls -r | more

​ cat /etc/passwd | wc

​ cat /etc/passwd | grep lwchen

​ ls -l | grep “^d”
